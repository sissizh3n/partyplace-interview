# Rules for Claude Agent


Business Rules (Validation)
                                                      
  1. Weekend minimum budget — bookings on             
  Saturday/Sunday require budget >= $1,500            
  2. Guest count required for large venues — if no    
  guest count provided, warn that results may not be  
  accurate. User should still be able to send the request, and the warning should be shown before and after form submission.
  3. Date must be in the future — reject past dates
